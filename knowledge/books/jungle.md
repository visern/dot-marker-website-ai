# Dot Markers — Animals Vol. 2

Part of the Jungle Animals Series, this is the sequel to the original
"Dot Markers — Animals" book. Sloths, lions, monkeys, and more jungle animals
each get their own big dot pattern to fill in.

It's the follow-up kids keep asking for after finishing the first Animals
book — more animals, more pages, same easy-to-fill oversized dots designed
for little hands.

Recommended for ages 2 and up. Sold in paperback on Amazon and on Etsy.

For the exact list of every animal that appears and which page it's on, see
the page-by-page index for this book (drawn directly from the interior
file) rather than this description — this file covers the book generally,
not every animal in it.
